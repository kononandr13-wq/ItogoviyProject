# Список референсов 

# Что пользователь может делать на веб-приложении
На главной странице отображается менюшка с кнопками: Объявить в розыск, Выписать штраф, База данных граждан, База данных сотрудников, База данных штрафов и розыска. Регистрация/Авторизация, рабочие систематические звания, подтверждение действительности сотрудников администрацией УФСБ, увольнение сотрудников.

# Как веб-приложение выглядит
## Главная страница.
Header
Внос граждан в розыск, выдача штрафов, увольнение сотрудников.

## Страница с подтверждением действительности сотрудников УФСБ.
Форма с полями: Ф.И.О сотрудника, почта, номер телефона, позывной, кнопка "Удалить".
