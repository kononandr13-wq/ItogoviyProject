# ItogoviyProject
# Список референсов


# Что пользователь может делать на веб-приложении
Объявлять в розыск граждан, выписывать штрафы, просматривать личное дело граждан, увольнять сотрудников(доступ с 9-го порядкового звания), выговоры(доступ с 9-го порядкового звания), повышать/понижать звание сотрудников по кнопке, выдавать доступ к сайту.

# Как веб-приложение выглядит 


## Главная страницаx
Главная страница - Кнопки: Объявить в розыск, выписать штраф, список сотрудников, список граждан

## Страница добавления сотрудников 
Подтверждение действительности сотрудников, выдача доступа к определенным функциям сайта.
